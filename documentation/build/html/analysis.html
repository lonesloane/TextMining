extract_indexes(files_index_filename=None, files_dates_index_filename=None, topics_occurrences_index_filename=None, topics_index_filename=None, only_highly_relevant=False)¶

Analyze the corpus, builds and saves index files

This function saves the following indexes:

  • files index: for each file (i.e. JT000XXX.xml), list of (topic, relevance)
  • topics occurrences index: for each topic, list of (file, relevance)
  • topics index: for each topic, (english label, french label)
  • file dates index: for each file, the date of publication
Parameters:
  • only_highly_relevant (bool) – set to true to extract only highly relevant topics
  • topics_index_filename (str) – name of the file for the topics index
  • topics_occurrences_index_filename (str) – name of the file for the topics occurrences index
  • files_dates_index_filename (str) – name of the file for the file dates index
  • files_index_filename (str) – name of the file for the files index
